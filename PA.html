Public Announcement

Public Announcement is a brand consultancy and media co based in New York City. I worked on some fun, wacky features for their homepage. Visit the website.

Hifi

I helped Public Announcement debut their first cover songs through the PA homepage. Using the Web Audio API, I developed the music visualizer and the hover events that activate certain audio queues.

Keyboard Navigation

The website can be navigated by using the keyboard arrows. A graphic pops up as the user presses the ↑ ↓ → ← keys.

Screensaver

After a certain amount of idle time, the homepage will enter Screensaver Mode. This constantly flips through the links on the home page. It can also be activated by using the ‘s’ key.

DeepTracking™

We track user activity like page visits through proprietary cookies to display fun, personalized messages.
